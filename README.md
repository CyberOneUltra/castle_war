## 📖 游戏简介

大陆名埃尔德兰，曾由七座王城共治，以"源火"维系各城繁荣。

三百年前，北方的烬落王发现了源火的黑暗面——吞噬生命即可壮大自身。他以此铸造不死军团，自封暗主，坐镇黑暗王座。

你是翠谷城的新任领主。前任领主在上一场战役中阵亡。翠谷城是联盟中最后一座还在运转的城池。

一路打过去，推翻黑暗王座。

## 🎮 游戏玩法

- 训练不同兵种，组建军队
- 释放法术，扭转战局
- 升级城堡，强化防御
- 击败 Boss，推进剧情
- 15 个关卡 × 3 个难度章节

## 🎯 操作说明

- **点击** 兵种按钮 → 训练士兵
- **点击** 法术按钮 → 点击战场释放
- **双击** 开场界面 → 播放 CG 动画
- **Esc** → 暂停 / 返回菜单

## 🛠️ 技术栈

- 纯 HTML5 Canvas + JavaScript，无框架依赖
- Web Audio API 音频系统（带 fallback）
- jsDelivr CDN 资源加速
- localStorage 存档 + Cookie 备份
